# Retail Sales Analysis Dashboard

## Overview
This project analyzes retail sales data to identify trends in regions, product categories, and time-based performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)

## Key Analysis
- Sales performance by region
- Top-performing product categories
- Monthly sales trends and seasonality
- Sales performance by category

## Key Insights
- The West region recorded the highest sales contribution.
- The technology category generated the highest revenue.
- Sales show seasonal peaks across specific months.
